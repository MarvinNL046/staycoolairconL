STAYCOOL AIRCO WEBSITE DOCUMENTATIE

1. OVERZICHT
============
StayCool Airco is een professionele website voor een airconditioning installatiebedrijf in Limburg. De website is gebouwd met React, TypeScript en Tailwind CSS, en biedt uitgebreide informatie over airconditioning producten en diensten.

2. TECHNISCHE STACK
==================
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion (animaties)
- React Router DOM (routing)
- React Helmet Async (SEO)
- EmailJS (contactformulieren)
- Lucide React (iconen)
